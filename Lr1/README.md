# Лабораторна робота №1  
## Встановлення Ubuntu та підготовка середовища для програмування мовою C

---

## Мета роботи

Ознайомитися з процесом налаштування середовища Ubuntu для програмування, встановити необхідні інструменти розробки та створити першу програму мовою C.

---

# 1. Оновлення системи

Після встановлення Ubuntu необхідно оновити систему та всі встановлені пакети.  
Це забезпечує стабільність, безпеку та коректну роботу програм.

У терміналі виконано:

```bash
sudo apt update
```

Команда:
- `sudo` — виконує команду з правами адміністратора
- `apt update` — оновлює список доступних пакетів із репозиторіїв

```bash
sudo apt upgrade
```

Команда:
- `apt upgrade` — встановлює нові версії програм та системних компонентів

---

# 2. Встановлення драйверів

Для автоматичного встановлення рекомендованих драйверів використано:

```bash
sudo ubuntu-drivers autoinstall
```

Команда:
- `ubuntu-drivers autoinstall` — автоматично встановлює відповідні драйвери для обладнання

Після цього систему було перезавантажено:

```bash
sudo reboot
```

Команда:
- `reboot` — перезавантажує систему для застосування змін

---

# 3. Встановлення інструментів для програмування

Для роботи з мовою C необхідний компілятор GCC та інші інструменти.  
Було встановлено пакет:

```bash
sudo apt install build-essential git -y
```

Команда:
- `build-essential` — містить компілятор GCC, бібліотеки та інструменти для компіляції
- `git` — система контролю версій
- `-y` — автоматичне підтвердження встановлення

Після цього система готова до розробки програм мовою C.

---

# 4. Створення першої програми

Для створення файлу використано текстовий редактор nano:

```bash
nano hello.c
```

Команда:
- `nano` — текстовий редактор у терміналі
- `hello.c` — назва майбутнього файлу

У редакторі було введено програму:

```c
#include <stdio.h>

int main() {
    printf("Hello Ubuntu\n");
    return 0;
}
```

Пояснення програми:
- `#include <stdio.h>` — підключення стандартної бібліотеки введення/виведення
- `main()` — головна функція програми
- `printf()` — виведення тексту на екран
- `return 0;` — завершення програми без помилок

---

# 5. Збереження файлу в nano

У редакторі nano:

1. Натиснути **Ctrl + O** — зберегти файл  
2. Натиснути **Enter** — підтвердити назву  
3. Натиснути **Ctrl + X** — вийти з редактора  

---

# 6. Компіляція програми

Після збереження файл було скомпільовано:

```bash
gcc hello.c -o hello
```

Пояснення:
- `gcc` — компілятор мови C
- `hello.c` — вихідний файл
- `-o hello` — створює виконуваний файл з назвою hello

Якщо помилок немає — компіляція завершена успішно.

---

# 7. Запуск програми

Для запуску виконуваного файлу введено:

```bash
./hello
```

Команда:
- `./` — означає запуск файлу з поточної директорії
- `hello` — назва виконуваного файлу

---

# 8. Результат виконання

Очікуваний результат у терміналі:

```
Hello Ubuntu
```
