# Лабораторна робота №7
## Тема роботи
Дослідження, моделювання та нестандартні підходи до аналізу процесів, файлових систем, безпеки та ресурсів в Linux.
## Мета роботи
Ознайомитися з основами системного програмування в середовищі Linux, навчитися працювати з файловою системою, каталогами, правами доступу, виконанням зовнішніх команд та вимірюванням часу виконання програм.
Отримати практичні навички написання програм мовою C для реалізації аналогів стандартних UNIX-утиліт та простих системних інструментів.

У ході виконання лабораторної роботи було досліджено основні принципи системного програмування в середовищі Linux. Було реалізовано низку програм мовою C, які працюють з файловою системою, каталогами та правами доступу.
